# 日报格式对比分析

## 现有 analyzer.py 日报

### 结构
1. 📊 价值投资日报
2. **概览**：今日新增、有效分析、无效文章、市场覆盖
3. **优先级分类**：🔴必读 / 🟡值得关注 / 🔵参考
4. **相关股票（按市场分组）**：A股/港股/美股/日股
5. **操作参考**：核心观点提炼 + 关键风险
6. **文章详情**：按优先级排序展示
7. **今日总结**：主要话题、热门股票

### 特点
- 基于文章质量评分（字数、关键词、核心观点数量）
- 按市场分组股票
- 提供操作参考（买卖建议倾向）
- 有无效文章过滤（<200字）

## 知识引擎日报

### 结构
1. 📊 价值投资日报（知识引擎版）
2. **概览**：今日新增来源数量
3. **实体投资信号**：📈📉➡️ 方向 + 置信度 + 摘要 + ⚠️Caveats
4. **新提及实体**：区分"有投资信号"vs"仅被提及"
5. **新提及概念**：跨实体通用认知
6. **来源详情**：标题、作者、链接、信号
7. **今日总结**：信号实体数、提及实体数、提及概念数

### 特点
- 基于结构化信号（direction/confidence/caveats）
- 实体为中心（而非文章为中心）
- 概念跨实体聚合
- 信号时间线可追溯
- Caveats 风险提示

---

## 对比分析

| 维度 | analyzer.py | 知识引擎 |
|------|-------------|----------|
| **核心组织** | 文章 | 实体 |
| **筛选逻辑** | 质量评分（字数/关键词） | 信号提取（方向/置信度） |
| **股票分组** | ✅ 按市场（A/港/美/日） | ❌ 无（可添加） |
| **优先级** | ✅ 必读/值得关注/参考 | ❌ 无（可添加） |
| **操作参考** | ✅ 核心观点提炼 | ✅ 结构化信号 |
| **风险提示** | ⚠️ 关键风险（文本） | ⚠️ Caveats（结构化） |
| **概念聚合** | ❌ 无 | ✅ 跨实体概念 |
| **可追溯性** | ❌ 单篇孤立 | ✅ 时间线聚合 |
| **数据驱动** | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ |

---

## 结论

### 知识引擎优势
1. **信号可追溯**：同一实体多篇来源聚合，展示观点变化
2. **结构化风险**：Caveats 标准化，便于后续量化分析
3. **概念网络**：跨实体通用概念，避免重复阅读
4. **数据驱动**：direction/confidence 可统计、可验证

### 待补充功能
1. **市场分组**：按 A/港/美/日 分组展示实体
2. **优先级**：基于信号置信度和实体重要性分级
3. **无效过滤**：<2000 字的文章在 ingest 时已过滤

### 建议
- **短期**：知识引擎日报作为补充视角（实体信号）
- **长期**：融合两者优势，analyzer.py 逐步退役
