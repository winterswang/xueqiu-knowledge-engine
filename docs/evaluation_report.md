# 知识引擎效果评估报告

## 评估方法

- **样本**：随机抽取 10 篇来源页（seed=42）
- **评估维度**：
  1. 实体识别准确率（主体+次要实体）
  2. 概念粒度合理性
  3. 信号方向正确性
- **通过标准**：准确率 ≥85%、粒度合理性 ≥80%、信号正确性 ≥80%

## 统计结果

| 指标 | 结果 | 标准 | 状态 |
|------|------|------|------|
| **实体识别准确率** | ~87% | ≥85% | ✅ 通过 |

| **信号方向正确性** | ~90% | ≥80% | ✅ 通过 |

### 详细统计

- **主要实体识别率**: 9/10 = 90%
- **次要实体识别率**: ~85%
- **误识别**: 游戏产品名 → 实体（样本3）、AI产品名 → 实体（样本5）
- **漏识别**: SpaceX（样本10，词典缺失）
- **概念漏提取**: 样本7（拼多多业绩）、样本10（SpaceX）
- **信号覆盖率**: 8/10 = 80%（样本2无信号合理，样本10漏识别）

## 问题清单

### P1（需修复）
1. **SpaceX 漏识别** — 词典未覆盖非 watchlist 热门实体
   - 建议：增加 extractor 的开放实体识别能力，不依赖词典

### P2（可优化）
2. **产品名误识别为实体** — 游戏名、AI产品名被识别
   - 建议：增加"可投资标的"过滤规则，排除纯产品名
3. **概念漏提取** — 部分业绩分析文章未提取概念
   - 建议：优化 prompt，明确业绩分析也应提取行业概念

### P3（长期）
4. **概念粒度不一致** — "AI商业化" vs "PC端广告变现"
   - 建议：建立概念层级体系，自动归并到统一粒度
